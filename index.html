About This Tool

PhishGuard Lite is a front-end cybersecurity project built to demonstrate secure UI design, clean JavaScript, and realistic phishing URL heuristics in a single-file application.

What it checks

The analyzer looks for risky signals such as IP-based links, suspicious brand impersonation, punycode, excessive subdomains, very long URLs, obfuscated characters, and suspicious host patterns that are often used in phishing campaigns.

How scoring works

Each suspicious signal contributes weighted points based on severity. The final risk level is mapped to clear categories so the result feels more like a practical triage decision than a random warning.

Important limitation

This tool analyzes URL structure only. It does not inspect page content, certificates, DNS records, or threat intelligence feeds, so it should be treated as an educational screening layer rather than a final verdict.
